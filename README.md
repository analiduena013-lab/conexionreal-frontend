# Conexión Real 💙

## 📌 Descripción del proyecto

**Conexión Real** es una aplicación web desarrollada con **React JS** orientada al agendamiento y gestión de citas médicas. El proyecto busca facilitar la interacción entre pacientes y profesionales de la salud mediante una interfaz sencilla, organizada e intuitiva.

La aplicación permite mejorar el proceso de solicitud, organización y consulta de citas médicas, brindando una solución tecnológica que optimiza la gestión de la atención en salud.

---

## 🎯 Objetivo del proyecto

Desarrollar un componente frontend utilizando React JS, aplicando buenas prácticas de programación, diseño de interfaces y herramientas de control de versiones, con el propósito de construir una solución web funcional para la gestión de citas médicas.

---

## 🚀 Tecnologías utilizadas

* **React JS**: Desarrollo de componentes e interfaz de usuario.
* **JavaScript**: Lógica y funcionamiento de la aplicación.
* **HTML5**: Estructura del contenido web.
* **CSS3**: Diseño y estilos visuales.
* **Vite**: Herramienta utilizada para la creación y ejecución del proyecto React.
* **Node.js**: Entorno para la gestión de dependencias.
* **Git y GitHub**: Control de versiones y almacenamiento del proyecto.

## ⚙️ Instalación y ejecución del proyecto

Para ejecutar el proyecto localmente se deben seguir los siguientes pasos:

### 1. Clonar el repositorio

```bash
git clone https://github.com/analiduena013-lab/conexionreal-frontend.git
```

### 2. Ingresar a la carpeta del proyecto

```bash
cd conexionreal-frontend
```

### 3. Instalar las dependencias

```bash
npm install
```

### 4. Ejecutar la aplicación

```bash
npm run dev
```

Después de ejecutar el comando, la aplicación estará disponible en el navegador mediante la dirección indicada por Vite.
